# Quick Start - Frontend Only

## Start the Frontend UI

Run this command in your terminal:

```bash
cd "/Users/machine/My Drive/Github Projects/Atlantic eSim/frontend" && npm run dev
```

The frontend will start on: **http://localhost:3001**

## What You'll See

- The Atlantic eSIM dashboard UI
- Login page (use mock data for now)
- Navigation and components

## Note

The backend API is not running, so some features won't work yet. But you'll see the full UI and can navigate through the interface.

To stop: Press `Ctrl+C` in the terminal
