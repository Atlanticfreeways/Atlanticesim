🔍 DEBUGGING STEPS
═══════════════════════════════════════════════════════════════════════════

1. OPEN DEVELOPER CONSOLE
   Mac:     Cmd + Option + I
   Windows: F12 or Ctrl + Shift + I
   
2. LOOK FOR ERRORS
   - Check the "Console" tab for any red error messages
   - Check the "Network" tab to see if assets are loading
   - Check the "Elements" tab to see if the DOM is being rendered

3. WHAT TO LOOK FOR
   ✅ Should see: "main.tsx loaded"
   ✅ Should see: "Root element: <div id="root"></div>"
   ✅ Should see: "Creating React root..."
   ✅ Should see: "React root created, rendering App..."
   ✅ Should see: "App rendered successfully"
   
   ❌ If you see errors, they will help identify the issue


📋 COMMON ISSUES & SOLUTIONS
═══════════════════════════════════════════════════════════════════════════

Issue 1: "Cannot find module" errors
  Solution: Clear browser cache (Cmd+Shift+R or Ctrl+Shift+R)

Issue 2: "React is not defined" errors
  Solution: Check if React is imported in the file

Issue 3: "Cannot read property of undefined" errors
  Solution: Check if dependencies are properly installed

Issue 4: Blank white page with no errors
  Solution: Check if CSS is loading (Network tab)

Issue 5: "Root element not found" error
  Solution: Check if index.html has <div id="root"></div>


🛠️ QUICK FIXES TO TRY
═══════════════════════════════════════════════════════════════════════════

1. HARD REFRESH BROWSER
   Mac:     Cmd + Shift + R
   Windows: Ctrl + Shift + R

2. CLEAR BROWSER CACHE
   - Open DevTools (F12)
   - Right-click refresh button
   - Select "Empty cache and hard refresh"

3. CHECK NETWORK TAB
   - Open DevTools (F12)
   - Go to Network tab
   - Refresh page
   - Look for failed requests (red)
   - Check if CSS and JS files are loading

4. CHECK CONSOLE TAB
   - Open DevTools (F12)
   - Go to Console tab
   - Look for any error messages
   - Copy error message and search for solution


📊 WHAT SHOULD HAPPEN
═══════════════════════════════════════════════════════════════════════════

When you open http://localhost:3000/:

1. Browser loads HTML
2. HTML loads CSS and JavaScript
3. JavaScript runs main.tsx
4. React mounts to #root element
5. App component renders
6. LandingPage component displays
7. You see blue gradient background with "Atlantic eSIM" title


🎯 IF STILL BLANK
═══════════════════════════════════════════════════════════════════════════

Try these steps:

1. Check if server is running:
   lsof -i :3000

2. Check server logs:
   tail -f logs/combined-*.log

3. Restart server:
   pkill -f "npm run start:dev"
   npm run start:dev

4. Check if frontend built correctly:
   ls -la frontend/dist/

5. Check if assets exist:
   ls -la frontend/dist/assets/
